## Sobre o projeto

Este é um quiz interativo desenvolvido em **Python**, inspirado no universo de **Elden Ring**.

O projeto tem como objetivo praticar lógica de programação, estruturas condicionais e interação com o usuário via terminal.

---

## Funcionalidades

- ✔️ Início com confirmação (S/N)
- ✔️ Entrada do nome do jogador
- ✔️ Perguntas de múltipla escolha
- ✔️ Sistema de pontuação
- ✔️ Resultado final personalizado

---

## 🛠️ Tecnologias utilizadas

- Python 3
- Lógica de programação
- Estruturas condicionais (if/else)
- Input do usuário

---

## 📌 Objetivo

Este projeto foi desenvolvido para treinar lógica de programação e criação de sistemas interativos no terminal.

---

##  Possíveis melhorias futuras

- Sistema de porcentagem de acertos (ex: 80% de acerto)
- Sistema de perguntas aleatórias (ordem diferente a cada execução)
- Sistema de classificação (ex: Tarnished, Warrior, Elden Lord)
- Níveis de dificuldade (fácil, médio, difícil)
- Feedback detalhado no final (mostrar quais perguntas errou e acertou)
